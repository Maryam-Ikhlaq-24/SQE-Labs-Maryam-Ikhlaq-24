# ISO 25010 Quality Analysis

This folder contains the Software Quality Engineering lab work for Week 2.
The assignment focuses on evaluating two mobile applications:
Google Chrome and Microsoft Edge  using the ISO/IEC 25010 quality model.

Contents include:

- A detailed quality assessment report covering all 8 ISO 25010 characteristics
- A comparative analysis with weighted scoring
- Python script ("quality_analysis.py") used to generate the radar chart
- Radar chart image visualizing quality comparison

The evaluation is based on real-device testing (Samsung A16, Android 16) under consistent conditions.
